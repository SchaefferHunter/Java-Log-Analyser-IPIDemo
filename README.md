Java Log Analyzer Demo
A small Java 17 / Maven demo project that parses application log lines and counts how many INFO, WARN, and ERROR entries appear.

Overview
This project is intended as a lightweight example for evaluating a simple log analysis workflow in Java. It demonstrates basic text parsing, Maven-based build automation, and JUnit 5 tests.

Features
Counts INFO, WARN, and ERROR log entries
Uses a simple line-based parser
Includes a small command-line demo application
Provides JUnit 5 unit tests
Uses Maven for build and dependency management
Requirements
Java 17 or higher
Maven 3.8+
